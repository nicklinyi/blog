Ubuntu 22.04 LTS 下安装 小鹤双拼

安装

前言

双拼输入法能大幅提高打字的效率，这里主要介绍如何在Ubuntu 22.04 LTS上安装双拼输入法

安装

1
2
3
4

$ sudo apt install ibus
$ sudo apt install ibus-libpinyin
$ sudo apt install ibus-rime
$ sudo apt install librime-data-double-pinyin

安装好上述软件后，在~/.config/ibus下新建文件夹rime

1

$ mkdir ~/.config/ibus/rime

然后，新建两个文件~/.config/ibus/rime/default.custom.yaml和~/.config/ibus/rime/ibus_rime.custom.yaml。 其中~/.config/ibus/rime/default.custom.yaml的内容如下（下面代码中#后面的为注释部分）

1
2
3
4
5

patch:
    schema_list:
        - schema: double_pinyin_flypy # 小鹤双拼  
    menu: 
        page_size: 9 # 设置后选词的个数为9个

这段代码用于配置小鹤双拼方案，并设置候选词为9个。

~/.config/ibus/rime/ibus_rime.custom.yaml文件内容如下：

1
2
3

patch:
    style:
        horizontal: true

这段代码的目的是设置输入法候选词的显示方式为横向。

配置

设置好以后内容后，还需要在~/.bashrc中添加如下内容：

1
2
3
4
5

   export GTK_IM_MODULE=ibus
   export XMODIFIERS=@im=ibus
   export QT_IM_MODULE=ibus
   export LC_CTYPE=zh_CN.UTF-8
   ibus-daemon -d -x

然后需要设置如下内容，打开Settings->Keyboard->Input Sources添加Chinese->Chinese(Rime).

然后登出系统，再重新登入系统。
